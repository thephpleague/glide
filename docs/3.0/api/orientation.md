# Orientation

## Orientation `or`

Rotates the image. Accepts `auto`, `0`, `90`, `180` or `270`. Default is `auto`. The `auto` option uses Exif data to automatically orient images correctly.

~~~ html
<img src="kayaks.jpg?h=500&or=90">
~~~
